# Exporting Submissions

## Overview

Exporting submission data is a crucial feature that allows you to analyze responses at your convenience. Formail offers multiple flexible export options, enabling you to download submission data from either the form export page or directly from the submissions page.

## How To Export Submissions

### Export from the Form Export Page

<Steps>
<Step>
### Access the Form Export Page

Navigate to the form whose data you wish to export. Each form will have an 'Export' option that leads you to the form export page.

</Step>
<Step>
### Specify the Date Range

1. **Select Start Date**: Use the date picker to choose the start date for the submissions you want to download.
2. **Select End Date**: Similarly, select the end date to define the range for your export.

</Step>
<Step>

### Choose the File Format

- **JSON**: Opt for JSON if you need a structured, machine-readable format that integrates easily with most software applications.
- **CSV**: Choose CSV for a format that is easy to view and edit in spreadsheet applications like Microsoft Excel or Google Sheets.

</Step>
<Step>

### Download the File

Once you've specified the date range and selected your preferred file format, click the 'Download' button. The file will be prepared and downloaded to your device, allowing you to access the submission data offline.

</Step>
</Steps>
### Export from the Submissions Page
<Steps>
<Step>
### Select Submissions

On the submissions page, select individual or multiple submissions you wish to export.

</Step>
<Step>
### Export Data

Click on the 'Export' button. You will then have the option to choose between exporting the data as JSON or CSV files.

</Step>
<Step>
### Download the File

After selecting the format, your file will be prepared and downloaded automatically to your device.

</Step>
</Steps>

<Callout title="Considerations">
  **Data Privacy**: Always handle the exported data in compliance with data
  protection regulations applicable to your organization. 
  
  **Storage**: Be mindful of where you store the downloaded files, especially if they contain
  sensitive information.
  
  **Regular Exports**: Setting a regular schedule for
  exports can ensure that you always have the latest data for analysis.
  
  Exporting your data through Formail provides a reliable and efficient way to
  manage your submissions and gain insights from your forms.

</Callout>
